About the House Price Prediction Project

This project predicts house prices using machine learning algorithms, including Ridge Regression, Linear Regression, and Lasso Regression. It is built with Python, Pandas, NumPy, and Flask, and utilizes a fixed dataset of houses from Kaggle. The model predicts prices based on input features such as location, bedrooms, bathrooms, and square feet.

Key Features

  • Predicts prices based on user input
  • Supports essential features: location, bedrooms, bathrooms, and square feet
  • Utilizes multiple regression models for comparison

Technologies Used

This project is powered by the following tools and libraries:

  • Python (NumPy, Pandas)
  • Ridge Regression, Linear Regression, and Lasso Regression
  • Flask for backend integration
  • Kaggle dataset for training and testing
