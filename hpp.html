About the House Price Prediction Project

This project uses Ridge Regression and Linear Regression models to predict house prices. By analyzing features such as area, bedrooms, square feet, and bathrooms, the model achieved an impressive 85% accuracy.

Key Features

  • Predict prices based on user input
  • Supports key features: Area, Bedrooms, Bathrooms, and Square Feet
  • Data-driven analysis using machine learning

Technologies Used

This project is powered by the following tools:

  • Python (NumPy, Pandas)
  • Ridge Regression and Linear Regression
  • Data Visualization
