About the Student Management System

This project is a Student Management System where there are two types of users: Students and Administrators. The administrator can log in and manage student information by adding details like student name, location, email, father's name, password, phone number, class, and branch. Students, on the other hand, can log in to view their personal details, but they are not allowed to make any changes. Built using PHP, JavaScript, HTML, CSS, and SQL.

Key Features

  • Admin login to manage student data
  • Student login to view personal details
  • Admin can add student information including name, email, phone, class, and more
  • Students cannot change their data

Technologies Used

This project is powered by the following tools and libraries:

  • PHP for backend development
  • SQL for database management
  • JavaScript for dynamic functionality
  • CSS for styling and Bootstrap for responsive design
