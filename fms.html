About the Financial Management System

This project is a Financial Management System where users can log in, sign up, and track their expenses. Once logged in, users can add their expenses along with the date and amount. The expenses are saved in the system, allowing users to view, track, and analyze them over time. The system also provides a dashboard with graphical representation of the expenses. Built using PHP, JavaScript, HTML, CSS, and SQL.

Key Features

  • User login and signup functionality
  • Track expenses with date and amount
  • Visual representation of expenses in a graph
  • Responsive dashboard for managing finances

Technologies Used

This project is powered by the following tools and libraries:

  • PHP for backend development
  • SQL for database management
  • JavaScript for dynamic functionality
  • CSS for styling and Bootstrap for responsive design
